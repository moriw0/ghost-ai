キャンバスの表示上の問題:

- ボックスシャドウまたはエレベーションスタイルによってキャンバスが浮いて見える
- ボーダーやボーダーラジアスによってキャンバスらしくなく「カード」のような見た目になっている
- キャンバスとコンテナの背景色が一致していない
- キャンバスの点線背景パターンが存在しないか誤っている
- `z-index` や配置（`absolute`/`relative`）が浮いているように見える原因になっている

- ラッパー div のスタイル（角丸、シャドウ、エレベーションされた背景など）がキャンバスを UI カードとして扱っている

サイドバーの問題:

- 左右のサイドバーはキャンバスを押しつぶすのではなく、キャンバスの上に浮いている必要がある
- サイドバーは `position: fixed` または `position: absolute` で高い `z-index` を持ち、キャンバスがその背後まで完全に広がるようにする
- キャンバスの背景（点線パターン）が両方のサイドバーの下まで端から端まで見えるようにする
